SVG

createMotionPath()

Creates pre-defined Tween parameter objects that animate along an SVGPathElement's coordinates and inclination.

const { x, y, angle } = svg.createMotionPath(path);

Parameters

Name Type
path CSS selector | SVGPathElement

Returns

An Object with the following properties:

Name Type Description
x Tween parameter Map to the x coordinate of the path element
y Tween parameter Map to the y coordinate of the path element
angle Tween parameter Map to the inclination of the path element

createMotionPath() code example

import { animate, svg } from 'animejs';

const { x, y, angle } = svg.createMotionPath('path');

// Animate the transforms properties of .car the motion path values
const carAnimation = animate('.car', {
  x: x,
  y: y,
  rotate: angle,
  ease: 'linear',
  duration: 5000,
  loop: true,
});

// Line drawing animation following the motion path values
// For demo aesthetic only
animate(svg.createDrawable('path'), {
  draw: '0 1',
  ease: 'linear',
  duration: 5000,
  loop: true
});
